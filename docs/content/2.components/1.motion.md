<Description>

`Motion` can use `as` to render as any HTML and SVG element,or you can pass `asChild` to render a child component.

</Description>

## Import

```ts
import { Motion } from 'motion-v'
```

```vue
  <Motion
    as="div"
    class="w-10 h-10 bg-primary rounded-lg"
  />
```
